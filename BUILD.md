# PlugHub - Инструкция по сборке

## 🚀 Быстрая сборка (3 шага)

### Шаг 1: Откройте PowerShell/CMD и перейдите в папку проекта
```bash
cd C:\Users\podkop\Desktop\key\plughub
```

### Шаг 2: Установите/обновите зависимости
```bash
npm install
```

### Шаг 3: Соберите приложение
```bash
npm run dist
```

**Готово!** Будет создана папка `dist` с установщиком и портативным .exe файлом

## 🔧 Пошаговая инструкция

### 1️⃣ Подготовка

Откройте PowerShell (нажмите `Win + R`, напишите `powershell`, нажмите Enter)

```powershell
# Перейдите в папку проекта
cd C:\Users\podkop\Desktop\key\plughub
```

### 2️⃣ Установка зависимостей

```powershell
# Установите все необходимые пакеты
npm install
```

Это займет 2-5 минут в первый раз.

### 3️⃣ Сборка приложения

```powershell
# Соберите React приложение и создайте Electron бинарники
npm run dist
```

Это займет 3-10 минут (зависит от скорости компьютера).

### 4️⃣ Откройте папку с готовыми файлами

```powershell
# Откройте папку dist в проводнике
start dist
```

Там вы найдете готовые установщики!

## 💻 Использование готового приложения

### Вариант 1: Установить с помощью инсталлятора (РЕКОМЕНДУЕТСЯ)

1. Откройте `PlugHub-0.1.0-nsis.exe`
2. Следуйте инструкциям установки
3. Приложение установится и создаст ярлык на рабочем столе
4. Дважды щелкните ярлык для запуска

**Преимущества:**
- Стандартная установка как обычные программы
- Создает ярлыки
- Проще удалить через "Установка и удаление программ"

### Вариант 2: Запустить портативный .exe

1. Откройте `PlugHub-0.1.0-portable.exe`
2. Приложение запустится сразу без установки

**Преимущества:**
- Не требует установки
- Можно носить на USB флешке
- Моментальный запуск

## 🛠️ Полная процедура (с нуля)

Если у вас есть проблемы, используйте эту полную процедуру:

```powershell
# 1. Перейдите в папку
cd C:\Users\podkop\Desktop\key\plughub

# 2. Удалите старые сборки
Remove-Item -Recurse -Force dist -ErrorAction SilentlyContinue
Remove-Item -Recurse -Force dist-electron -ErrorAction SilentlyContinue

# 3. Очистите кэш npm
npm cache clean --force

# 4. Переустановите зависимости
Remove-Item -Recurse -Force node_modules -ErrorAction SilentlyContinue
npm install

# 5. Соберите приложение
npm run build
npm run electron-build
```

## ⚙️ Что происходит при сборке?

1. **TypeScript компиляция** (`tsc`)
   - Компилирует TypeScript в JavaScript

2. **Vite сборка** (`vite build`)
   - Собирает React приложение
   - Минифицирует CSS и JS
   - Создает папку `dist` с готовым приложением

3. **Electron Builder** (`electron-builder`)
   - Упаковывает приложение
   - Создает установщик NSIS для Windows
   - Создает портативный .exe
   - Подписывает приложение (если ключ задан)

## 📊 Размер сборки

- **Установщик NSIS**: ~150-200 MB
- **Портативный .exe**: ~180-220 MB
- **На диске после установки**: ~200-300 MB

(Размер зависит от node_modules)

## 🐛 Решение проблем

### Ошибка: "npm: The term is not recognized"
**Решение:** Перезагрузите PowerShell или используйте cmd.exe

### Ошибка: "electron-builder not found"
**Решение:** 
```powershell
npm install -g electron-builder
npm install
npm run dist
```

### Ошибка: "Port 5173 already in use"
**Решение:** Закройте другие процессы npm или используйте другой порт:
```powershell
npm run dev -- --port 3000
```

### Сборка зависает
**Решение:** Дайте больше времени или перезагрузитесь:
```powershell
npm run dist --verbose  # Покажет детали что происходит
```

## ✅ Проверка сборки

После сборки проверьте:

1. **Существует папка `dist`**
   - Должны быть файлы: `*.exe`, `*.msi` и т.д.

2. **Проверьте размер файлов**
   - Должны быть > 100 MB

3. **Запустите установщик**
   - Должен установиться без ошибок

4. **Запустите приложение**
   - Должно открыться окно с PlugHub

Для продакшена добавьте:

1. **Иконку приложения**

   - electron-updater будет автоматически обновлять

4. **Обновления версии**
   - Измените `version` в package.json
   - Пересоберите
