## 📦 СПОСОБ 3: .EXE установщик (ТРЕБУЕТ FIX)

Electron-builder имеет баг с macOS symbolic links на Windows. 

**Решение:**

Обновите electron-builder до последней версии:
```bash
npm install -g electron-builder@latest
npm install --save-dev electron-builder@latest
npm run electron-build
```

Или используйте облачное решение (GitHub Actions для сборки на Linux/macOS).
