## ⚠️ ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

### Проблема: "npm: command not found"
**Решение:** 
- Перезагрузите PowerShell
- Или используйте cmd.exe вместо PowerShell

### Проблема: "Port 5173 already in use"
**Решение:**
- Закройте другие окна npm
- Или перезагрузитесь

### Проблема: Сборка зависает
**Решение:**
- Дайте ей больше времени (15-20 минут)
- Если совсем не двигается - нажмите Ctrl+C и пробуйте снова
- Убедитесь что достаточно места на диске (минимум 2 GB свободного)

### Проблема: "electron-builder: command not found"
**Решение:**
```powershell
npm install -g electron-builder
npm install
npm run dist
```

## 💡 СОВЕТЫ

1. **Для большего файла:**
   - Сборка может быть 200-300 MB
   - Это нормально (Node.js + Electron очень тяжелые)

2. **Для более быстрой сборки:**
   - Установите SSD диск
   - Закройте ненужные программы
   - Дайте больше оперативной памяти

3. **Для обновления версии:**
   - Измените номер версии в `package.json`
   - Пересоберите
   - Версия автоматически обновится в установщике

4. **Для распространения:**
   - Используйте `PlugHub-0.1.0-nsis.exe` для пользователей
   - Залейте на GitHub Releases
   - Или на сайт для скачивания
